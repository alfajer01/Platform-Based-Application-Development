# Tutorial 5.1 - Networking

## Task Overview

> Project ini merupakan lanjutan dari project Flutter sebelumnya dan digunakan untuk menampilkan data product dari Laravel API menggunakan package `http`.

## Folder Structure

- `app/` : berisi project Flutter untuk tugas ini
- `screenshot/` : berisi hasil capture aplikasi

## How to Run

1. Pastikan Laravel API sudah berjalan dan bisa diakses di jaringan yang sama.
2. Masuk ke folder `app/`.
3. Jalankan perintah berikut:

```bash
flutter pub get
flutter run
```

4. Pada halaman utama, tekan tombol `Go to Tutorial 12`.
