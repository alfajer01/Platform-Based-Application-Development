# Tutorial 1.2 - Intro Dart

## Task Overview

> Buatlah program Dart yang menggunakan array 2 dimensi bertipe integer dengan jumlah baris 4, dan isilah masing-masing baris dengan perulangan. Buatlah juga fungsi Dart yang menerima dua nilai integer positif dan mengoutputkan nilai FPB (Faktor Persekutuan terBesar) dari dua bilangan tersebut.

## Folder Structure

- `app/` : berisi project Dart console untuk tugas ini
- `screenshot/` : berisi hasil capture program

## How to Run

1. Masuk ke folder `app/`.
2. Jalankan perintah berikut:

```bash
dart run
```

3. Masukkan dua bilangan positif saat diminta oleh program.
